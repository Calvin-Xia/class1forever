## 数据安全

本项目中的学生数据已从 GitHub 仓库中移除，通过 Cloudflare Pages 环境变量安全存储。

- `js/data.js` 已添加到 `.gitignore`，不会被提交到仓库
- 生产环境数据通过 Cloudflare Pages 环境变量 `STUDENTS_DATA` 注入
- 本地开发可保留 `js/data.js` 文件用于测试

详细部署说明请查看 [DEPLOYMENT.md](./DEPLOYMENT.md)。
